# ADDF Prompt Catalog

**Version:** 0.1.0  
**Framework:** ADDF 3.5

This catalog contains copy-ready prompts for ADDF sessions. Each prompt declares its mode, lists files to load first, provides the prompt text, and defines expected output.

## How to Use

1. Choose the prompt that matches the session goal.
2. Load the files listed in `Files to Load First`.
3. Copy the prompt text into the model session.
4. Replace bracketed placeholders.
5. Keep all durable project state in files, not chat memory.
