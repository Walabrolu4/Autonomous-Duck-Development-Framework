# v0.1 Release Notes - Mini Task Tracker

Write this file at release time.

**Version:** 0.1.0  
**Release name:** Core Task List  
**Date:** 2026-06-07

## Summary

Mini Task Tracker v0.1 provides a local-only browser task list for one operator.

## What's New

- Add tasks.
- Mark tasks complete and reopen them.
- Persist tasks in localStorage across browser reloads.

## Known Issues

Tasks can be lost if browser storage is cleared.
