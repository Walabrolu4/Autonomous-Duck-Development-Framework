# v0.1 Scope - Mini Task Tracker

## In Scope

| Item | Notes |
|---|---|
| Add task | Operator can create a task with a title. |
| Mark complete | Operator can mark an open task complete. |
| Reopen task | Operator can return a completed task to open status. |
| Persist to localStorage | TaskList survives browser reload. |
| View all tasks | Open and completed tasks are visible. |

## Out of Scope

| Item | Target |
|---|---|
| User accounts | Not planned |
| Server storage | Not planned |
| Sharing | Not planned |
| Filtering | Not planned |
| Search | Not planned |
